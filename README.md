宝贝时刻
===

记录你和宝宝的欢乐时光。 Inspired by @Ke.Li

开发说明
---

项目使用 [Carthage](https://github.com/Carthage/Carthage) 做依赖管理。

1. 安装 Carthage

    ```
    brew install carthage
    ```

2. 更新项目依赖

    ```
    carthage update --platform iOS
    ```

3. 运行单元测试: Cmd + U
